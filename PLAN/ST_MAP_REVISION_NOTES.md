# Revision Notes

The project plan and coding-agent prompt were updated to add:

- English-only public repository policy.
- English default and fallback locale.
- Separate text and image provider abstractions.
- Custom OpenAI-compatible text API support.
- Synchronous, asynchronous task-based, and generic REST image provider designs.
- Independent provider profiles for services such as DeepSeek-compatible text endpoints and Nano Banana Pro-style image endpoints.
- Optional world, region, city, building, room, and dungeon generation.
- Text-only, image-only, hybrid, and fully manual workflows.
- Credential-handling requirements.
- Blueprint schema validation.
- Unlabeled image-background requirements.
- Timeouts, cancellation, polling limits, and provider tests.
- A dedicated optional AI provider milestone.
- Updated release roadmap and security checklist.

Core map viewing and editing remain usable without any AI provider.
